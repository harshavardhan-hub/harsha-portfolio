## üõ† Tech Stack

### **Frontend**
- **Framework:** [React 18.2.0](https://react.dev/) with [Vite](https://vitejs.dev/)

- **WhatsApp Integration:** Automated notification messages
- **CORS:** Enabled for frontend-backend communication
